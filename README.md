# Chocolate Chip Cookies Recipe 🍪

## About
A simple recipe webpage built using HTML
as part of freeCodeCamp HTML practice.

## Features
- Recipe introduction with image
- Ingredients list
- Step-by-step instructions

## Tags Used
- Semantic tags — main, section
- Headings — h1, h2
- Image — src, alt
- Unordered list — ul, li (ingredients)
- Ordered list — ol, li (instructions)
- Paragraphs
